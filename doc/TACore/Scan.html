Overview

Note:

A group should not include a “beacon” (asset tag) / Iris id, only Gateway device id's should be used to search for proximity data. Using an Iris ID will result in a nil or empty return.

> Scan / Proximity Data.

The scan data returned is filtered for duplicates along with ordered for what device is most near in a given group. A group is defined by an array of Gateway device id's (Cirrus id's).

def self.by_cirrus(token, api_key, cirrus)
  request(:post, '/api/v1/scans', token, {:body => {:devices => cirrus}, :headers => {:client_api_key => api_key}})
end
